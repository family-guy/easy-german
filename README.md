## Description

Download [Easy German](http://easygerman.org/) videos and transcripts into Google Drive.

## Usage

`pip install easygerman`

then:

```
import easy_german

easy_german.get_videos(max_downloads=150, max_results_per_page=20)
easy_german.get_transcripts()
```
